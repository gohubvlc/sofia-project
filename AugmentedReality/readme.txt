The application has been programmed in Unity environment using the google SDK ARcore. The app is an example of what would happen in a real case. It shows a forest area, full of trees. At the moment a fire starts and it is detected by Sofia, arrows are displayed on the screens of the devices near the fire, indicating the safe route to escape from the fire.

Used technology:
Unity 2017.4.9f1
ARCore SDK v1.5.0
Android SDK 7.0 (level API 24)
